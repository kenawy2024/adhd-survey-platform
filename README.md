# منصة ADHD - Arabic ADHD Survey Platform

منصة عربية احترافية لاستبيانات اضطراب نقص الانتباه وفرط النشاط (ADHD)

## المتطلبات
- Node.js v18+
- MongoDB (محلي أو MongoDB Atlas)

## التثبيت والتشغيل

```bash
# 1. تثبيت الحزم
npm install

# 2. إعداد المتغيرات (راجع ملف .env)
# تأكد من تشغيل MongoDB

# 3. تهيئة قاعدة البيانات (مطلوب مرة واحدة)
npm run seed

# 4. تشغيل الخادم
npm start

# للتطوير مع إعادة التشغيل التلقائي
npm run dev
```

## الرابط بعد التشغيل
- الموقع: http://localhost:3000
- لوحة التحكم: http://localhost:3000/admin/login.html
- بيانات الدخول: admin / admin123

## هيكل المشروع
```
├── server.js           # الخادم الرئيسي
├── seed.js             # تهيئة البيانات
├── models/             # نماذج قاعدة البيانات
├── routes/             # مسارات API
├── middleware/         # المتوسطات
└── public/             # الواجهة الأمامية

    ├── survey.html     # صفحة الاستبيان
    ├── results.html    # صفحة النتائج
    ├── about.html      # عن ADHD
    ├── admin/          # لوحة التحكم
    ├── css/            # ملفات التنسيق
    └── js/             # ملفات JavaScript
```

## الاستبيانات المضمنة
1. مقياس تقرير الذات لاضطراب ADHD للبالغين (ASRS)
2. اختبار صعوبة الانتباه وتشتت التركيز
3. اختبار الاندفاعية وضبط النفس
4. اختبار فرط النشاط الحركي
5. اختبار تأثير ADHD على الحياة اليومية

## لوحة التحكم - المميزات
- إضافة/تعديل/حذف الاستبيانات
- بناء الأسئلة مع الإجابات والنقاط
- إدارة قواعد التصنيف
- عرض الإحصائيات والرسوم البيانية
- إدارة الإعلانات (AdSense + HTML مخصص)
- سجل الإجابات

## إعداد Google AdSense
1. اذهب للوحة التحكم > إدارة الإعلانات
2. أضف إعلاناً جديداً من نوع AdSense
3. أدخل Publisher ID وAd Slot الخاص بك
4. فعّل الإعلان
